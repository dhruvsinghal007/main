Simple CRUD application with accounts management. It contains an eclipse project and a war file. 
This is an eclipse project which needs to be imported in workspace. 

--The application has four tabs here---
1. Customers
2. Items
3. SalesAndPurchases
4. Payments

Each tab supports the CRUD(Create, Read, Update and Delete) functionalities. The project can be imported in eclipse.
The war file just needs to be placed in <tomcat installation dir>/webapps folder. 
Then start the tomcat server and type he url : http://localhost:8080/AccountingWebApp/ The application should work fine.
